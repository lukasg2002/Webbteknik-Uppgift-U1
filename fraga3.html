Fråga 3: CSS

a. Vad är en regel i CSS och vad består den av?

Svar: En CSS regel anger vilka element som ska påverkas och hur de ska påverkas.
En regel består av en selektor där man specificerar en eller flera element, och sedan klamrar med egenskaper där man kan val utav till exempel typsnitt, textstorlek eller färg.

b. Vad är skillnaden mellan id och class? När bör man använda id respektive class?

Svar: Skillnaden mellan id och class är att ett id är unikt och det kan bara förekomma en gång i html filen medan en class kan upprepas flera gånger i HTML-filen.
Id är bäst att använda när du bara vill ändra ett specifikt element i CSS medan class är bättre att använda ifall du vill återanvända en stil för flera element.
